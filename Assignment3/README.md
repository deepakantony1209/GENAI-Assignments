# Assignment 3 - Functions and Lambda Operations

## Overview

This assignment covers function definitions, lambda functions, recursive functions, and functional programming concepts like map and filter.

## Files

### 1. basicFunction.py

Demonstrates basic function definitions and usage.

- Function creation and calling
- Parameter passing
- Return values

### 2. lambdaFunction.py

Explores lambda functions and functional programming.

- GST calculator using lambda
- Map operations on lists
- Filter operations with predicates

### 3. recursiveFunction.py

Implements recursive function patterns.

- Base cases and recursive cases
- Problem solving with recursion

### 4. miniProblem.py

Interactive menu-driven program for price management.

- Add prices to a list
- Calculate average price
- Find highest price
- User-friendly menu system

## Running the Scripts

```bash
python basicFunction.py
python lambdaFunction.py
python recursiveFunction.py
python miniProblem.py
```

## Requirements

- Python 3.x
