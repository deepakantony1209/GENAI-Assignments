# Assignment 4 - File Operations and Data Processing

## Overview

This assignment focuses on working with file I/O operations in Python, reading and writing data to files, and processing sales and product information.

## Files

### 1. salesRecords.py

Writes sales data to a file.

- Creates/appends sales records to a file
- Formats data with comma-separated values
- Demonstrates file writing operations

### 2. appendNewSales.py

Appends new sales entries to existing records.

- Adds new sales data to the file
- Demonstrates append mode file operations

### 3. readFiles.py

Reads and displays file contents.

- Opens and reads sales or product files
- Demonstrates file reading operations

### 4. exportDiscountedPrices.py

Exports product prices with applied discounts.

- Calculates discounted prices
- Writes results to a new file

### 5. productInfo.py

Manages product information from files.

- Reads product data
- Processes and organizes product details

### 6. summaryReport.py

Generates summary reports from file data.

- Analyzes sales or product data
- Creates summary output

### 7. checkFilePath.py

Checks if files exist in the directory structure.

- Searches for files by name
- Provides file path information
- Searches recursively through subdirectories

## Running the Scripts

```bash
python salesRecords.py
python appendNewSales.py
python readFiles.py
python exportDiscountedPrices.py
python productInfo.py
python summaryReport.py
python checkFilePath.py
```

## Data Files

- `files/` - Directory containing input/output data files

## Requirements

- Python 3.x
- Input data files in the `files/` directory
